# CookMeThis

A meal planner application that uses a large recipe database. It allows users to select a list of ingredients and gives options for potential meals that can be prepared.

# Features

<ul>
  <li>Bookmarks</li>
  <li>Whitelist</li>
  <li>Blacklist</li>
  <li>Fuzzy Searching</li>
</ul>
